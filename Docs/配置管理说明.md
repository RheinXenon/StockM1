# AI Agent 配置管理说明

## 配置文件位置

配置文件保存在：`Agents_Experience/user_setting.ini`

## 如何保存配置

1. 在可视化页面中，点击 **AI Agent模拟炒股**
2. 在 **⚙️ 配置设置** 部分填写您的配置
3. 点击 **💾 保存配置** 按钮
4. 配置将自动保存到 `user_setting.ini` 文件（如果文件不存在会自动创建）

## 配置项说明

### API配置
- **api_base**: API服务地址
- **api_key**: API密钥（必填）
- **model**: 使用的模型名称
- **api_call_interval**: API调用间隔时间（秒）

### 模型参数
- **temperature**: 控制输出的随机性 (0.0-2.0)
- **max_tokens**: 每次生成的最大token数
- **history_window_days**: Agent可以查看的历史数据天数

### 系统提示词
- **system_prompt**: 定义Agent的角色、目标和交易策略

### 交易设置
- **initial_capital**: 模拟交易的初始资金
- **stock_pool**: 可交易的股票列表
- **start_date**: 模拟开始日期
- **end_date**: 模拟结束日期

## 配置优先级

1. 如果存在 `user_setting.ini` 文件，系统将加载该文件的配置
2. 如果文件不存在，系统将使用 `.env` 文件中的默认配置
3. 如果 `.env` 文件也不存在，系统将使用内置的默认配置

## 首次使用

首次使用时，您需要：
1. 配置 API Key（必需）
2. 选择至少一只股票
3. 设置模拟时间范围
4. 点击保存配置按钮

## 注意事项

- 配置文件使用 UTF-8 编码
- API Key 会以明文存储在配置文件中，请妥善保管
- 建议将 `user_setting.ini` 添加到 `.gitignore` 中，避免泄露敏感信息
- 不需要 `user_setting.ini.example` 文件，所有配置都通过网页界面管理

## 故障排除

如果保存配置失败：
1. 检查 `Agents_Experience` 目录是否有写入权限
2. 查看错误信息，了解具体失败原因
3. 确保所有必填项都已填写
4. 检查股票池是否至少选择了一只股票
